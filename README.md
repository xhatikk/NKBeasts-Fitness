# NKBEASTS Premium V6

Static professional fitness website prepared for Cloudflare Pages.

## Upload to GitHub
1. Extract this ZIP.
2. Upload all files to the root of the `NKBeasts-Fitness` repository.
3. Replace existing files.
4. Commit changes.
5. Cloudflare Pages will deploy automatically.

## Cloudflare Pages
- Framework preset: None
- Build command: empty
- Build output directory: `/`

## Important
Replace the placeholder `#` social links in `index.html` with your real YouTube, Instagram, Facebook and TikTok links.
The live RSS uses a public CORS proxy and falls back to three built-in cards if the proxy is unavailable.
