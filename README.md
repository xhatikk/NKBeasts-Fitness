# NKBEASTS V19 – Single File Fix

Ky version ka CSS dhe të gjithë JavaScript-in direkt brenda `index.html`.
Prandaj cookies, butonat, kalkulatorët, radio, navigimi dhe ikonat sociale nuk varen më nga folderi `js`.

Ngarko në root:
- index.html
- favicon.svg
- manifest.webmanifest
- ads.txt
- README.md

Ruaje patjetër këtë strukturë për RSS në Cloudflare Pages:
- functions/api/rss.js
